🍎 **AI-Powered Food Quality Detection & Inventory Management System**

An intelligent system that uses multiple cutting-edge AI models to detect, classify, and assess the freshness of food items in real-time.

### Key Features:
- 🔍 **Gemini Vision AI** - Accurate food identification and counting
- 🍃 **ResNet18 Classifier** - Fresh/Rotten classification with high accuracy
- 📦 **YOLOv8 Detection** - Real-time object detection for multiple food items
- 📝 **OCR Scanning** - Automated expiry date extraction
- 📊 **Analytics Dashboard** - Visual insights and statistics
- 📷 **Camera Integration** - Live webcam support
- ⚠️ **Smart Alerts** - Expiry notifications and alerts
